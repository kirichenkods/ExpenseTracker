# Expense Tracker

### Описание
Пользователь ведёт учет расходов и доходов.
Можно проверить баланс. Дополнительно можно проверить баланс с конвертицией в указанною валюту по актуальному курсу ЦБ.

Документация доступна по адресу ipaddress:8080/swagger-ui/index.html, например http://localhost:8080/swagger-ui/index.html
